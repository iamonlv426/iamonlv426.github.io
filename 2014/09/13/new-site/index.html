New site

13 Sep 2014

Welcome to my new site, which is just a little bit different from the old one. Like my old WordPress beast, this site retains a very minimalistic design. I'd like to say that's because I aspire to a plain, almost spartan chic in my web layouts. But the truth is it's hard enough to put something this plain together. Maybe I'll build on the basic design--more or less inspired by Zach Holman--to create something a little more filled out.

Right now, the site looks incredibly empty on a big screen, but works really nicely on smartphones and tablets (thank you Bootstrap!).

The masthead is the same as the old one, mostly because I'm just that uncreative. I've also dumped a lot of the stuff I included in my old site like the article sections, the endless RSS options, and the section collecting my tweets.

Instead, at least for now, this is going to house blog posts and that's it. Anyone looking for links to my work, contact details, etc. can find everything they need on the about page.

What I'm really excited about with this new site design are the changes under the hood. WordPress served me well, but I wanted to create something a little more nimble. Enter Jekyll, a static site generator that takes plain text files as its input and creates a static site.

Right now I'm hosting my Jekyll site on GitHub, which means I'm using Git version control to track changes to this site. It's very geeky, but also a lot of fun.

Will all these new, more basic tools inspire me to blog more regularly? I don't know.

But playing around with this new set of tools should keep me going for at least a few weeks.
